Initializing:
1)  config->keys.js:
    Enter the Database and Email credentials
2)  Run db.sql to create the Database

Running:
1)  node app.js
2)  In the browser enter -> http://localhost:3000

Functionality:
You'll be in the Company homepage and have four options:
- Admin Login
- Employee Registration
- Package Status Update:
  - Adding new Package and sending Email to client
  - Updating location of existing package using QRcode
- Tracking Package using Maps
